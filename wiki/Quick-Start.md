# 🚀 Démarrage rapide / Quick Start

---

<summary><b>🇫🇷 Français</b></summary>

## En 5 minutes

### 1. Télécharger & Extraire

[📦 Télécharger PyMkvPropEdit Portable](https://github.com/Crysisjim/PyMkvPropEdit/releases/latest)

Extraire le ZIP → double-cliquer sur `run.bat`

### 2. Ajouter des fichiers

- Onglet **Input** → **"Add Files"** ou **"Add Folder"**
- Ou glisser-déposer les fichiers MKV directement

### 3. Configurer les pistes

- Onglet **Audio** → Cocher "Edit Track Audio 1" → Définir langue et nom
- Onglet **Sous-titres** → Même chose pour les sous-titres
- Onglet **Général** → Optionnel : titre, suppression des tags

### 4. Traiter

Cliquer sur **"Process Files 🚀"** (bouton vert en bas)

Une fenêtre résumé s'affiche avec succès/erreurs. Une notification Windows 11 apparaît.

### Cas d'usage courants

**Renommer les pistes audio d'une série :**
- Onglet Audio → Track 1 : `jpn (Japanese)`, nom = `Japonais FLAC`
- Track 2 : `fra (French)`, nom = `Français AAC`
- Traiter tout le dossier

**Ajouter une jaquette :**
- Onglet "Image de couverture" → Parcourir → Sélectionner l'image
- Traiter

**Synchroniser le doublage :**
- Onglet "Audio Sync 🎵" → Charger le MKV → Analyser → Appliquer

## In 5 minutes

### 1. Download & Extract

[📦 Download PyMkvPropEdit Portable](https://github.com/Crysisjim/PyMkvPropEdit/releases/latest)

Extract the ZIP → double-click `run.bat`

### 2. Add files

- **Input** tab → **"Add Files"** or **"Add Folder"**
- Or drag & drop MKV files directly

### 3. Configure tracks

- **Audio** tab → Check "Edit Track Audio 1" → Set language and name
- **Subtitles** tab → Same for subtitle tracks
- **General** tab → Optional: title, tag deletion

### 4. Process

Click **"Process Files 🚀"** (green button at the bottom)

A summary window shows success/errors. A Windows 11 notification appears.

### Common use cases

**Rename audio tracks for a series:**
- Audio tab → Track 1: `jpn (Japanese)`, name = `Japanese FLAC`
- Track 2: `fra (French)`, name = `French AAC`
- Process the entire folder

**Add cover art:**
- "Cover Image" tab → Browse → Select image
- Process

**Sync dubbing:**
- "Audio Sync 🎵" tab → Load MKV → Analyze → Apply
